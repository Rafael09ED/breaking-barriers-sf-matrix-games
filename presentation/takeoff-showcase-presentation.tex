\begin{frame}{Why this helps}
  \begin{itemize}
    \item You test understanding by making decisions inside the scenario.
    \item You see how incentives move the world in different directions.
    \item You practice argument quality, not just opinions.
    \item As far as I know, no one else has built this exact combo:
      Engle matrix format + AI alignment scenario + LLM and human play.
  \end{itemize}

  \vspace{0.4em}
  {\color{Muted}\small This is for exploration and training, not forecasting.}

  \source{Curry \& Price (2014); Griffin \& Riggs (2024); Hogan \& Brennen (2024)}
\end{frame}
\note{
  Keep novelty phrasing cautious: ``as far as I know.'' This avoids overclaiming while
  still making the differentiation clear.
}
