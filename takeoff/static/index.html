How the game works

  1. 01Choose a role

    You control one actor. Four persona-conditioned LLMs control the others.

  2. 02Make one argument

    State one action, the result you want, and one to three reasons it should work.

  3. 03Face the umpire

    The umpire weighs your support against concrete risks and sets a modifier.

  4. 04Change the world

    Code rolls 2d6. Success or failure commits new facts that constrain every later move.
